Seguridad

Privacidad por diseno

No es una promesa de marketing. Es la arquitectura real del sistema.

🔒 Infraestructura

  • Servidor VPS privado — sin nubes compartidas
  • Autenticacion por Telegram User ID unico
  • Cero almacenamiento en servicios de terceros
  • PM2 como gestor de procesos con auto-restart
  • Deploy via git pull — sin CI/CD externo

🗃️ Flujo de Datos

  • 4 archivos de contexto cargados por mensaje
  • Re-indexacion automatica despues de cada conversacion
  • Logs episodicos diarios en Markdown
  • Busqueda FTS5 local — sin APIs de busqueda externas
  • Latido proactivo cada 180 minutos con deduplicacion
